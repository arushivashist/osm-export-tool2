About

This platform allows you to create custom OpenStreetMap exports for various HOT regions. You can specify an area of interest and a list of features (OpenStreetMap tags) for the export. A current OpenStreetMap data extract for that area in various data formats is then created for you within minutes.

How to Get Started

You can view and search through existing HOT exports at our Exports page. You do not need an account to view or search for existing exports. If you want to create an export and you are a new user, please register for an account. If you already have an account with HOT, please log in using the log in form on this page. More detailed help is available on our help page.
