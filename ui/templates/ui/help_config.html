Configuration File Help

The user can upload configuration files in the tool to apply to the data selected for export. This help page will provide guidance on the types of configuration files, how to search for them on the 'Store' page and upload them on the 'Create' page.

File Types

Preset Files

Transform Files

Transform files are used to execute arbitrary SQL (spatialite) commands on the selected exported data.
For example, create new database columns, drop existing database columns, rename columns, merge columns, or drop records.

Example 1. Insert new column into the planet_osm_point table and fill it with a copy of the "amenity" value:

alter table planet_osm_point add column testcolumn varchar; update planet_osm_point set testcolumn=amenity;

Example 2. Insert two new columns into tables building_roof_material_stone and building_roof_material_concrete, setting them to 1 if the existing column building_roof_material contains these terms:

alter table planet_osm_polygon add column building_roof_material_stone integer; alter table planet_osm_polygon add column building_roof_material_concrete integer; update planet_osm_polygon set building_roof_material_stone=1 where building_roof_material like "%stone%"; update planet_osm_polygon set building_roof_material_concrete=1 where building_roof_material like "%concrete%";

Language Files

Language files are used to translate the contents of a database field from one language into another.
For example the term "bakery" may be transformed into "boulangerie" or "road" to "jalan".

Upload Files

Files can be uploaded on the 'Select Configuration' tab of the 'Create' page. As it is part of the wizard that takes the user through creating a new export, it can only be accessed once the previous steps have been completed.

The 'Select Features' tab provides the user with a text box to enter a 'Filename' and a 'Configuration Type'. Once these have been completed, the user can choose to either search their desktop for the file by clicking 'Browse Desktop' or select a file from the configuration file Store by clicking 'Browse Store'.

Once the configuration file has been selected, the user will be brought back to the 'Create' wizard where they can choose to publish the file globally by clicking 'Publish File'. {% endblock %}
