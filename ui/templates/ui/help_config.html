Configuration File Help

The user can upload configuration files in the tool to apply to the data selected for export. This help page will provide guidance on the types of configuration files, how to search for them on the 'Store' page and upload them on the 'Create' page.

File Types

  • Preset: A JOSM Preset.

Upload Files

Files can be uploaded on the 'Select Preset' tab of the 'Create' page. As it is part of the wizard that takes the user through creating a new export, it can only be accessed once the previous steps have been completed. The 'Select Preset' tab provides the user with a text box to enter a 'Filename'. Once this has been completed, the user can choose to either search their desktop for the file by clicking 'Upload Preset' or select a file from the configuration file Store by clicking ''.

Once the configuration file has been selected, the user will be brought back to the 'Create' wizard where they can choose to publish the file globally by clicking 'Publish File'.
