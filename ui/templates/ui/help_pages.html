Page Navigation Help

The OSM Export Tool has three main sections; an area to create new exports (Create), an area where existing exports are stored and can be re-run (Exports), and an area where configuration files are stored and can be accessed (Store). This help page will take the user step by step through these three sections, providing information for the available functions and how to use them:

Create

User must be registered and logged in to create a new export.

1. Describe Export: Enter the name, description and event of the new export, select the area of interest on the map (right section) and click 'Next'.
2. Export Formats: Select one or more file format(s) for the export and click 'Next'.
3. Export Features: Expand and select feature tags from the HDM or OSM tree list for export and click 'Next'. Presets can not be chosen if tags have been selected in previous tab.
4. Configuration File: Upload a configuration file by locating the file and specifying a name and type. Choose whether to save it to your personal account and/or publish it to the global store for everyone to access and click 'Next'.
5. Export Summary: This provides an overview of the export settings. The option to publish the export globally for everyone can be selected. Click 'Create Export' to complete and run the new export.
6. Export Details: Contains the status and summary of the export, as well as the history of runs and options to either re-run or clone the job.

Exports

Existing exports can be searched, viewed and re-run from the Exports page. An account is not needed view or search for existing exports.

On the Exports page there are the following filter options:
1. Search Box: Enter Name/Event/Description keywords to filter.
2. Start Date/End Date: Filter based on date of creation.
3. Interactive Map: Visually displays location of filtered exports and can also be used to filter geographically by selecting on the map.
4. My Exports: Displays personal exports if the user is logged in.
5. Next Page: Move on to display more results.

Store

Saved configuration files can be searched, viewed and downloaded from the Store page. An account is not needed to filter and search the saved configuration files.

On the Store page there are the following filter options:
1. Search Box: Enter Name/Event/Description keywords to filter.
2. Start Date/End Date: Filter based on date of creation.
3. File Type: Filter based on File Type.
4. My Configuration Files: Displays personal configuration files if the user is logged in.
5. Next Page: Move on to display more results.
