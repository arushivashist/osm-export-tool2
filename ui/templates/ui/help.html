Help Documentation

The OSM Export Tool allows users to create custom OpenStreetMap exports for selected HOT regions. This help documentation is split into the following sections to assist users to navigate the site, use the functions effectively and successfullly create and run exports.

Page Navigation

This section takes the user through the three main areas of the OSM Export Tool website and provides step by step directions on what functions are available and how to use them:
  • Create
  • Exports
  • Store

Export File Formats

This section takes the user through the five file formats available for selection to export the OSM data into, as well as provide useful links for users to find further information on their application:
  • Esri SHP
  • Garmin IMG
  • Google KMZ
  • OSM PBF
  • SQlite SQL

Feature Tag Selection

This section takes the user through the feature selection functionality built into the Export Tool, which allows the user to select OSM tags to export specific map features for export:
  • Humanitarian Data Model
  • OpenStreetMap Data Model

Configuration Files

This section takes the user through the configuration files that can be used with the Export Tool to provide extra functionality:
  • Presets
